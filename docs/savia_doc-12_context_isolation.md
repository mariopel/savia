- Resumen del Concepto: Bifurcación de Hilos de Trabajo (Context Isolation)
Para mantener la "mente" de la IA enfocada y no diluir la calidad de las respuestas, los proyectos complejos deben dividirse en chats (hilos) temáticos. La instalación de un sistema operativo subyacente (WSL2) es una tarea de "Capa 1" (Hardware/OS), mientras que este chat es de "Capa 7" (Aplicación/Criptografía).

- Justificación Técnica y Evidencia:
Mezclar contextos de infraestructura local (Windows/Ubuntu) con código de producción (Docker/Payload) en una misma ventana de chat aumenta exponencialmente la probabilidad de que la IA sufra "alucinaciones" o pierda el hilo de las variables de entorno de SAVIA.

- Configuración / Lógica Acordada (Hoja de Ruta Inmediata):
1. Pausa Táctica: Congelamos este chat exactamente aquí. Quedará como tu enciclopedia criptográfica y de Traefik.
2. Nuevo Entorno: Abres un chat completamente nuevo. Puedes iniciarlo diciendo: *"Soy Mario. Necesito que actúes como experto en Windows 11 y me guíes paso a paso, desde cero, para instalar WSL2 con Ubuntu, VS Code y Git"*.
3. El Comando de Oro: Para que vayas ganando tiempo, la instalación en Windows 11 se ha simplificado a un solo comando. Deberás abrir "Windows PowerShell" (haciendo clic derecho -> Ejecutar como Administrador) y tipear: `wsl --install`.
