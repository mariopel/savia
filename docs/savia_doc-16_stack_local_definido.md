- Resumen del Concepto: "Local Development Stack" (Stack Local)
Para lograr la simetría de entornos y habilitar el desarrollo asistido por IA (GitOps), la lista de compras exacta y definitiva de software que instalaremos en tu PC (Oficina/Casa) es la siguiente:

1. Windows 11: El sistema operativo anfitrión (Host).
2. WSL2 (Ubuntu): El subsistema nativo de Linux que correrá por debajo para máxima compatibilidad con las llaves criptográficas.
3. Docker Desktop: El motor gráfico y de fondo para levantar los contenedores de SAVIA simulando la VM de producción.
4. Google Antigravity: El Entorno de Desarrollo Integrado (IDE). Reemplaza al VS Code clásico, aportando la lectura del protocolo MCP y los agentes autónomos.
5. Git: El motor invisible de control de versiones que Antigravity usará para sincronizar tu computadora con GitHub.

- Justificación Técnica y Evidencia:
Al utilizar Antigravity en lugar del VS Code estándar, eliminamos la necesidad de configurar extensiones complejas de IA a mano. El agente leerá tu terminal de WSL2 y tus contenedores de Docker Desktop directamente desde la misma ventana.

- Configuración / Lógica Acordada:
El usuario comprende la lista de requerimientos y procederá a realizar la instalación y configuración paso a paso en un chat/hilo separado para preservar la higiene del contexto (Context Isolation).
